🎓 5 заданий с localStorage

💾 Задание 1: Сохранить любимый цвет

Введите цвет, нажмите кнопку и цвет сохранится в localStorage

📊 Задание 2: Счетчик посещений страницы

Счетчик увеличивается при каждой загрузке страницы

📝 Задание 3: Запомнить текст заметки

Напишите заметку, нажмите сохранить, текст восстановится при перезагрузке

👋 Задание 4: Кнопка "Скрыть приветствие"

Скройте приветствие, оно останется скрытым после перезагрузки

Добро пожаловать на сайт!

🛒 Задание 5: Мини-список покупок

Добавляйте товары в список, он сохранится в localStorage
